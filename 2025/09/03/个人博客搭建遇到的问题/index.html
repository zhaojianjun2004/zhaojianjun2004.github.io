个人博客搭建遇到的问题

关闭ssh博客无法访问的问题

这说明你目前是通过 在终端前台运行 hexo s 来启动博客的。一旦你断开 SSH 连接（Terminus 关闭），这个进程就会被终止，博客自然就无法访问了。

根本原因：hexo s 是一个前台进程，它依赖于你的 SSH 会话。会话一关闭，进程就被杀死了。

✅ 正确的解决方案：使用后台服务运行

你需要让博客作为一个后台服务（daemon）运行，不依赖于终端会话。以下是几种推荐的方法：

🛠 方法一：使用 nohup + &（最简单，适合临时使用）

nohup 可以让命令忽略挂起信号（HUP），即使终端关闭，进程也不会终止。

  2. 使用 nohup 启动 Hexo 服务：

    • nohup：忽略终端关闭信号。
    • --ip 0.0.0.0：允许外部访问。
    • > hexo.log 2>&1：将输出日志重定向到 hexo.log 文件。
    • &：在后台运行。

screen 和 tmux 是终端复用工具，可以创建会话，即使断开连接，会话仍在运行。
